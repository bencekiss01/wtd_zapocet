# wtd_zapocet

This project is a simple static website created as a graded assigment for the web technologies course.

## Projekt description

The website contains multiple pages created with html, css and javascript.

Pages included:
-Home
-About
-Gallery
-Form
-Data
-Contact

## Features

-Navigation between pages
-Gallery with image preview
-Registration form with clients-side validation
-Data page with json loading using ajax
-Custom css styling

## Technologies used

-HTML
-CSS
-JavaScript

Live demo: https://bencekiss01.github.io/wtd_zapocet/
